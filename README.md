# 📊 Econometrics Notebooks | Portafolio de Econometría

## 🇪🇸 Español

### 📌 Descripción

Este repositorio contiene una colección de notebooks desarrollados en Google Colab enfocados en el estudio y aplicación de métodos econométricos.

El objetivo es construir un portafolio sólido que combine teoría económica, estadística y programación aplicada con Python.

---

### 🎯 Objetivos

* Aplicar modelos econométricos clásicos
* Analizar datos reales con herramientas de Data Science
* Interpretar resultados estadísticos en contexto económico
* Desarrollar habilidades en análisis cuantitativo

---

### 🧠 Contenido

* Regresión lineal simple
* Regresión múltiple
* Supuestos del modelo clásico (OLS)
* Heterocedasticidad
* Series de tiempo *(en progreso)*

---

### 🛠️ Tecnologías utilizadas

* Python
* pandas
* numpy
* matplotlib
* statsmodels
* Google Colab

---

### 🚀 Cómo usar este repositorio

1. Clonar el repositorio:

```
git clone https://github.com/tu-usuario/econometrics-notebooks.git
```

2. Abrir los notebooks en:

* Google Colab
* Jupyter Notebook

---

### 📈 Ejemplo de análisis

Cada notebook incluye:

* Definición del problema
* Exploración de datos
* Implementación del modelo
* Interpretación de resultados

---

### 🤝 Contribuciones
